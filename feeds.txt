# AIニュース 購読ソース一覧
# ------------------------------------------------------------------
# 書き方:
#   ・1行に1つ。「表示名|RSSのURL」の形式（表示名は省略可）
#   ・末尾に |filter を付けると、AIに関係する記事だけに絞り込みます
#     （GIGAZINEやASCIIのような“総合ニュース”ソース向け）
#     付けない場合は全記事を取り込みます（AI専門ソース向け）
#   ・先頭が # の行はコメント（無視されます）
#   ・不要なソースは行頭に # を付ければ無効化できます
#   ・増やしたいときは下に追記してください
#
#   ★ URLの代わりに「gnews:キーワード」と書くと、そのキーワードの
#     Googleニュース検索結果を集めます（公式RSSが無い企業・話題向け）。
#     例:  トヨタのAI|gnews:トヨタ 生成AI
#
#   ★ 「#cat:分野名」と書くと、それ以降のソースがその分野（色バッジ・
#     画面上のフィルタに使用）に分類されます。
# ------------------------------------------------------------------
