# 🤖 AIニュースまとめアプリ（第1段階）

AI関連の主要ニュースをRSSで集めて、スマホで見やすい1枚のページにまとめます。
現在は **APIなし・GitHubなし** で、PCだけで動く構成です。

## 使い方

2つの使い方があります。お好みでどうぞ。

### A. アプリとして使う（おすすめ・更新ボタン付き）
`アプリ.bat` をダブルクリック。
ブラウザが開き、**その場で全ソースを集めて一覧表示**します。
画面右上の「更新」ボタンでいつでも最新を取り直せます。
（黒いウィンドウを閉じるとアプリは停止します）

### B. 1枚のHTMLファイルにする（サーバー不要）
`更新.bat` をダブルクリック。
ニュースを集めて `output\index.html` を作り、自動でブラウザで開きます。
このHTMLは単独ファイルなので、後でスマホに渡すのに向いています。

### コマンドで動かす場合
```
python app_server.py   # アプリ版（http://127.0.0.1:8770/）
python news_app.py     # HTML書き出し版（output\index.html）
```

## ニュースのソースを変えたいとき

`feeds.txt` を編集します（メモ帳でOK）。
- 1行 = 1ソース。「表示名|RSSのURL」の形式
- 行頭に `#` を付けると、その行は無効になります
- 増やしたいときは下に追記するだけ

## ファイル構成

| ファイル | 役割 |
|---|---|
| `news_app.py` | 収集の本体（集める→整える）。HTML書き出しもここ |
| `app_server.py` | アプリ版サーバー（開くたびに集めて表示） |
| `feeds.txt` | 購読するニュースソースの一覧 |
| `アプリ.bat` | アプリ版を起動する入口 |
| `更新.bat` | HTML書き出し版を実行する入口 |
| `output/index.html` | 書き出し版で出来るページ |

## これからの拡張（メモ）

- **第2段階：スマホで見る** … `output` を OneDrive や GitHub Pages 経由でスマホに表示
- **第3段階：自動化** … Windowsのタスクスケジューラで毎朝自動実行
- **AIモード** … Anthropic APIキーを用意すると、`news_app.py` の `make_summary()` を
  Claude要約に差し替えて、日本語3行要約・重要度タグ・重複まとめが可能になります
  （本体の他の部分は変えずに差し替えられる設計です）
